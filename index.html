About the site:

Mobiletest.it is a free resource to testing mobile responsive web pages.

The goal of mobiletest.it is to provide a simple and intuitive framework for testing site performane on a variety of mobile devices.

In additional to the phone emulator - I work to provide current data on the most common device resolutions and modern best practices for responsive development.

If you enjoy using this free tool - feel free to check out the GitHub, or some other projectes at tylermaran.com
